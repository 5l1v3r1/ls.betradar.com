How to install?
===============

```
$ git clone --recursive git@github.com:mahendrakalkura/ls.betradar.com.git
$ cd ls.betradar.com
$ mkvirtualenv --python=python3 ls.betradar.com
$ pip install --requirement requirements.txt
```

How to run?
===========

```
$ cd ls.betradar.com
$ workon ls.betradar.com
$ python run.py
```
